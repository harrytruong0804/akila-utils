# akila-utils

Claude Code plugin marketplace for AI agent development.

## Quick Start

```bash
# 1. Add the marketplace
/plugin marketplace add harrytruong0804/akila-utils

# 2. Install a specific plugin
/plugin install openai-agents-tracing@akila-utils
/plugin install context-engineering@akila-utils
/plugin install plugin-dev@akila-utils
/plugin install akila-confluence@akila-utils
```

## Plugins

| Plugin | Description |
|--------|-------------|
| `openai-agents-tracing` | Self-hosted tracing dashboard for OpenAI Agent SDK |
| `context-engineering` | Context engineering audit for AI agent codebases |
| `plugin-dev` | Plugin development lifecycle — dev, version, release, self-update |
| `akila-confluence` | Publish to Confluence with images/attachments — render figures, upload, embed |
